# Heartdisease-prediction
# ❤️ Heart Disease Prediction System

## 📌 Project Overview

This project is a Machine Learning model that predicts whether a person is at risk of heart disease based on medical parameters.

It is built using Python and scikit-learn.

---

## ⚙️ Technologies Used

* Python
* pandas
* numpy
* scikit-learn
* matplotlib
* seaborn
* Jupyter Notebook

---

## 📊 Dataset

The dataset contains medical features such as:

* Age
* Sex
* Chest Pain Type
* Blood Pressure
* Cholesterol
* Maximum Heart Rate
* and more...

---

## 🤖 Machine Learning Model

This project uses classification algorithms such as:

* Logistic Regression
* Random Forest
* Decision Tree

---

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Heartdisease-prediction.git
```

### 2. Create virtual environment

```bash
python -m venv venv
```

### 3. Activate environment

```bash
venv\Scripts\activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run notebook or script

```bash
jupyter notebook
```
